なぜアメリカ投資を行うのか

なぜアメリカ投資を行うのかというと、自分の資金をすべて円で持っていることに不安を感じたからである。

日本は人口減少時代が始まってしまった。

いまさら人口減少に手を打ってももう遅い。

まあ何も手を打たないよりはましだけど。

人口が減少すると、その国の経済は低下する。

その国で消費されているもののかなりの部分が自分の国で作られているからだ。

なぜ人口が減少しているのかというと

・こども一人当たりにかかるお金の問題

・日本に対する将来の不安（借金等）

の二点が挙げられると思う。

いくらこども一人当たりにかかるお金の問題を解決しても、

日本の抱えている借金をどうにかしないとどうにもならない。

首相は人が減っていくのだから、それに対応して

日本をどう変えていくかのビジョンを示さなくてはならない。

小泉首相にはそれは出来ない。

よって日本は沈没していく一方だろう。

商品の値段を上がらざるを得ない

商品の値段は需要が多ければ上がるので、

中国、インドや他の発展途上国が発展していくうえでは避けては通れない。

原油はこれだけ値段が上がれば、今まで採掘をあきらめていた油田でも取れるようになると思うので、当分使えなくなることはないと思うのだが、当然値段はどんどん上がっていく。

そこで、原油の代わりとして期待されているのが天然ガスだと思う。

天然ガスは石油のように中東に集中していないという特性を持つが、

石油前提で成り立っている部分で天然ガスをどう使えるようにするかが問題なんだろうなー
