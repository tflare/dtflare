Fri, May 1, 2009
Operaクッキーの扱い

ブラウザを使っていると使用しないクッキーがたまってくる。

昔は時々削除していたけど

以下の設定により削除する必要がなくなった。

Tools > Preferences > Advanced > Cookies > Delete new cookies when exiting Operaをオンにして

クッキーが必要なサイトで右クリックからEdit Site Preferences

Cookies > Delete new cookies when exiting Operaをオフにする。

すると必要なサイトだけクッキーを受け入れることができる。

そのためクッキーを削除する必要がなくなる。

Fri, Dec 22, 2006
Operaを9.10にアップデート

Ubuntu6.10でOpera(static)を9.10にアップデート 

$sudo gedit /etc/apt/sources.list

で開いて以下を追加

deb http://deb.opera.com/opera stable non-free

そして以下を実行すると 

$ sudo apt-get update

W: GPG error: http://deb.opera.com stable Release: 公開鍵を利用できないため、以下の署名は検証できませんでした: NO_PUBKEY 033431536A423791
W: これらの問題を解決するためには apt-get update を実行する必要があるかもしれません

上記メッセージが出たため以下を実行 

$ gpg --keyserver pgp.nic.ad.jp --recv-keys 033431536A423791
$ sudo gpg --armor --export 033431536A423791 | sudo apt-key add -

そして以下を実行すると無事インストールできた。 

$ sudo apt-get install opera-static
