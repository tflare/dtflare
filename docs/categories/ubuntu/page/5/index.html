Thu, Nov 20, 2008
Ubuntu8.10　起動時間短縮

Ubuntu8.10にアップデートしてから起動が遅くなった。（２分前後かかるようになった）

そこでbootchartを使って測定してみると

dhclient-scriptの実効に時間がかかっていることが分かった。

このため

/etc/network/interfaces の

iface eth0 inet dhcpを以下に変えることによって

iface eth0 inet static

45秒まで短縮できた。

ここまで時間がかかるのはおかしいので環境の問題のような気もする。

起動が5回に1回ぐらいしか成功しない状態なのでこれで改善できればうれしい。

Opera snapshot

opera_static-9.50-1983.gcc4-qt4_i386.deb

を入れたときはまともに動いていたのが

opera-static_9.50-1992.gcc4-static-qt3_i386.deb

を入れたあとにブックマークが化けて表示されるようになった。（メニューを日本語にするとメニューも化ける）

フォントの設定を変えても駄目

opera-static_9.50-1997.gcc4-static-qt3_i386.deb

その後上記Build1997を入れても改善しない。

opera_9.50-1997.gcc4-qt4_i386.deb

そこでqtの問題かと思いqt4のバージョンを入れ直したら直った。

結局opera-staticでBuild1992からqt4がなくなったので

static-qt3バージョンを入れたのが原因だった。

staticじゃないoperaのqt4版で上記問題が解決し

他の問題も発生していないのでこれで作業する予定。

Wed, Oct 17, 2007
日本語入力

Ubuntu(scim-anthy)で日本語入力の切替えが出来ない事があって悩んでいたんだけど、

CapsLock有効であることが原因であると気づいて

調べてみたら以下で解決した。

https://lists.ubuntu.com/archives/ubuntu-jp/2006-November/000257.html

https://lists.ubuntu.com/archives/ubuntu-jp/2006-November/000259.html

/etc/scim/config

~/.scim/config

の以下のCapsLockを削って

/Hotkeys/FrontEnd/ValidKeyMask = Shift+Control+Alt+CapsLock+Meta

システム→システム管理→言語サポートで英語を選択してログインしなおすとOKのようだ。
