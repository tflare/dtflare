zsh

chsh

でログインシェルを/bin/zshに変更。

いまさらながらとっても便利

.zshrcを以下に設定

設定内容は以下を参照するとわかりやすい。

http://www.ayu.ics.keio.ac.jp/~mukai/translate/zshoptions.html 

# compinitの初期化
autoload -U compinit
compinit
# 補完候補カーソル選択
zstyle ':completion:*:default' menu select=1
## ヒストリ関連
# 履歴ファイル
HISTFILE="$HOME/.zsh_history"
HISTSIZE=100000
SAVEHIST=100000
# 時刻を記録
setopt extended_history
# 履歴をインクリメンタルに追加
setopt inc_append_history
# 履歴の共有
setopt share_history
# 直前と同じコマンドラインはヒストリに追加しない
setopt hist_ignore_dups
##　プロンプト関連
setopt prompt_subst
local BLUE=$'%{\e[34m%}'
local DEFAULT=$'%{\e[m%}'
PROMPT=$BLUE'${USER}@${HOST}%(!.#.$)'$DEFAULT
RPROMPT='[%~]'

Fri, Dec 22, 2006
Operaを9.10にアップデート

Ubuntu6.10でOpera(static)を9.10にアップデート 

$sudo gedit /etc/apt/sources.list

で開いて以下を追加

deb http://deb.opera.com/opera stable non-free

そして以下を実行すると 

$ sudo apt-get update

W: GPG error: http://deb.opera.com stable Release: 公開鍵を利用できないため、以下の署名は検証できませんでした: NO_PUBKEY 033431536A423791
W: これらの問題を解決するためには apt-get update を実行する必要があるかもしれません

上記メッセージが出たため以下を実行 

$ gpg --keyserver pgp.nic.ad.jp --recv-keys 033431536A423791
$ sudo gpg --armor --export 033431536A423791 | sudo apt-key add -

そして以下を実行すると無事インストールできた。 

$ sudo apt-get install opera-static

Ubuntu6.10でデスクトップにホームアイコンを表示させる方法

コマンドラインからgconf-editorを立ち上げる。
