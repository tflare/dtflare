エネルギー系以外への商品投資

食糧関係やエネルギーを除いた商品等にETFを使って投資したい

それで探してみたんだけど見つかったのが

DJ-AIG Ex-Energy

http://www.etfsecurities.com/en/document/downloads/ETFS_Ex-energy_Fact_Sheet.pdf

DJ-AIG Agriculture

http://www.etfsecurities.com/en/document/downloads/ETFS_Agriculture_Fact_Sheet.pdf

上記のイギリスに上場している物しか見つからなかった。

ヨーロッパには商品個別ごとに買えるETFがあるらしい(参考：ETF Watch)

ヨーロッパにしか上場していないのでは買えないので

商品系INDEXを探してみると

エネルギーの割合が少なく、農産物の割合が多いのが

Dow Jones AIG Commodities Index

(参考：Rogers International Commodity Index TRAKRS vs. Other Commodity ETFs)

これのETFであるiPath DJ AIG Commodity ETN(Symbol:DJP)を買うのが良いかな。

問題はDow Jones AIG Commodities Indexのリターンがあまりよくないこと。

とりあえずDJ-AIG Ex-Energyのようなのが、出るまで待ちかな。

UbuntuをWindowsXPと比較したときの個人的な利点・欠点

利点

ソフトのアップデートが楽

/etc/apt/sources.listを修正しておけば

後はアップデートでOKを押す手間ですむ

書庫マネージャが結構使える。

いろいろな圧縮形式に対応し

sudo apt-get install unrar

とするとRAR形式にも対応してくれる等結構使える。

OSのインストールが楽

WindowsXPよりもUbuntuの方がOSのインストールが楽

欠点

Operaが時々固まる

たくさんタブを開いた状態で右クリックをして何かを選択すると発生する。

システムモニタを見るとOperaのCPU使用率が上がっているのがわかる。

Operaで文字列入力ができなくなるときがたまにある。

一度ログアウトしてからログインしなおすと復帰する。

銀行系のサービスでInternetExplorer必須の所が使えない。

証券会社でダメだったことはないけど、銀行系の場合ブラウザチェックで引っかかる。

原油と天然ガス

原油価格が上がっている。

原油価格は今後数年間は上がると思う。

なぜかというと以下の原因が上げられると思う。

・原油に対する投資が需要に対して足りていない

・既存の油田の産出量の低下

・新規の高コストの油田に頼らざるを得ないこと

（石油天然ガス・金属鉱物資源機構資料を参照）

現在の原油高騰の一因として

投資資金の流入も上げられるが、投資資金の流入がなくなり原油価格が下がると

油田を運営している会社で廃業するところが出てきて

長期的に原油価格が上がるという傾向は変わらないのではないかと思う。

（実際に投資資金の流入がなくなる事はあまり考えられない。

それは株と商品は逆に動くことが多いからだ。

株が上がるとき、商品は下がり

株が下がるとき、商品は上がる

傾向にある。

株と商品を組み合わせることによって、

リスクが下げられ、リターンの上昇が見込まれるからだ。）

原油価格の高騰が続くと原油から天然ガスのシフトが本格的に

始まるのではないかと思う。

ただ天然ガスにも問題があり、そう簡単にはシフトできない。

なぜかというと原油は液体なのに対し、天然ガスは気体だからだ。

原油は液体なので輸送コストがやすいという利点がある。

それに対し天然ガスは気体なのでパイプラインでつなぐかNGLにするかしなくてはいけない。

これは初期投資がかなりかかることを意味している。

天然ガスは石油よりも二酸化炭素排出量が少なく、埋蔵量も多い

天然ガスのシフトは遅れざるを得ないが

パイプラインさえ引ければなんとかなるので

産出地と近ければシフトは進むだろう。
