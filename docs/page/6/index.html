トイレの底の掃除

トイレの底の掃除を以下の方法でやった。

灯油用スポイトがあれば楽かも

1.　トイレの水を止める

　　　参考:http://www4.ocn.ne.jp/~cleanpet/newpage63.html

2.　ゴミのプラスチック製品で水をすくう

　　全てはすくえないので適切なところで切り上げる

3.　サンポールを入れる。

4.　3〜4時間待つ

5. 　ブラシで洗う

これで大体きれいになる。

これでもきれいにならなかったら

スプーンを使って削る。（前回は使った）

サンポールを増やす。

4の時間を増やす

ことで対処しよう。

Opera設定まとめ

現在のOpera設定のまとめ

Tools→Preference→Advanced(ツール→設定→詳細設定)

Content(コンテンツ)

JavaScript Options（JavaScriptオプション）

User JavaScript Folder(ユーザ JavaScript フォルダ)設定

Style Option（スタイルプション）

10.70だとフォントの設定がうまくいかないところがあるのでここで設定

History(履歴)

Check if cached page is updated on the server(ページの更新をサーバーに確認するタイミング)

Check documents(ドキュメント) Every 24 hours(24時間に1度)に変更

Check images(画像) Every Week(1週間に1度)に変更

表示高速化、画像は変わる可能性は少ないので1週間に1度とする。

Cookies

Cookie Delete new cookies when exciteing opera(Operaの終了時にCookieを削除する) オンに変更

クッキーは後で削除するのが大変なので

必要なものだけサイトごとの設定で、このチェックをオフとする。

これによりクッキーの削除が不要となる。

opera:config

opera:config#UserPrefs|TrustServerTypes オンに変更

これがないと左クリックしたときにダウンロードされず

内容が表示されてしまう場合がある。

オフだとサーバの設定を使用するようだが、

サーバの設定がきちんと行われていないところがそれなりにあるので

オンとする。

AutoPagerize

AutoPagerizeを導入

http://d.hatena.ne.jp/os0x/20071202/oautopagerize

次へ等をクリックしなくても、次のページが表示される。

ものすごく便利

Ubuntu起動時に以下を実行（システム→設定→自動起動するアプリから追加）

sh ‘/home/taki/AutoAutoPagerize.sh’

起動時に実行することで、定期的に最新化を行う。

/home/taki/AutoAutoPagerize.shの中身

wget http://ss-o.net/userjs/0AutoPagerize.SITEINFO.js

mv 0AutoPagerize.SITEINFO.js /home/taki/user_javascript/

au携帯料金節約、お得情報

一度考えた上で行動を起こせば、あとはなにも意識しなくても節約やお得となる方法を行う。

いちいち考えなくてはいけない方法で行うと、行わなくなってしまう可能性があるのでよくない。

この考えに基づいて自分が行った、au携帯料金節約、ポイント取得等のお得情報をまとめる。

1.auプレミアメンバーズ

無料で1年経過ごとに加算のポイント（10年目以降なら1年で1000ポイント）、

毎月のご利用額に応じたポイント等がつく

無料なので入るべき。

http://www.au.kddi.com/point_after/service/premium/index.html

2.シンプルプラン

au買い方セレクト前の携帯電話を持っていたので

安い携帯電話を買ってシンプルプランにして

携帯電話を今までのものを使うようにした。

携帯電話は5,000円ぐらいで買えた（ポイント充当でまかなった）

プランSSからプランSSシンプルに変更で

基本料が1,620円ぐらいから780円ぐらいになり840円安くなった。

5000÷840＝5.95 6ヶ月で元が取れます。

シンプルコース用料金プラン

http://www.au.kddi.com/ryokin_waribiki/ryokin/simple/index.html

フルサポートコース用料金プラン

http://www.au.kddi.com/ryokin_waribiki/ryokin/plan/index.html

3.パケット代節約

EZ設定→表示設定→添付データ→再生しない

を行い画像を表示しないようにする。

画像はサイズが大きいのですぐに料金が上がるため効果的。

しかも画像広告をカットできる。

いつもよく行くサイトがあれば一度添付データ再生するでアクセスしたあとに

再生しないとすれば画像がキャッシュされて再生しないにしたあとも表示される。

これで必要なところはカバーできる可能性もある。

4.パケット代節約（ダブル定額解約）

これは上記と違って意識しなくてはならないが

効果が大きいため記載する。

ダブル定額を契約すると

無料通話料をパケット通信の料金に使えなくなるので

無料通話が余ることがあった。

そこでダブル定額を解約し、無料通話内でパケット通信を行うこととした。

解約前までは上限4,410円/月をいつも使っていたが

この利用用途は主に電車の中でのWeb閲覧であったので、

これを家でPCで行うようにすれば4,410円が浮くことになると考え実行。

電車の中では浮いたお金で何かをするようにすれば良いと考えた。

これにより4,410円節約。

3つの3.0インタフェース

PCの拡張性に大幅な影響を与えるのが

USB,Serial ATA,PCI-Expressの3つである。

この3つにはアップデートが迫っている。

USB,Serial ATAについては拡張ボードの導入で

対処可能な部分があるが、マザーボードが新しいバージョンを考慮した設計になっていないと

能力が十分に生かせないという問題がある。

そのため出来れば新しいバージョンを考慮した設計がなされているものを導入したい。

PCI-Express 3.0については

一般ユーザにとって主な用途となるビデオカード(GPU)が

PCI Express 2.0で十分と思われることもあり、あまり気にしなくても良いと考える。

そのためUSB 3.0、Serial ATA 3.0について考慮した設計がなされ、

チップセット内蔵された時が買い時と考える。

つまりIntelなら2012年、AMDなら2011年と考える。

USB 3.0

チップセット内蔵でマザーボードに搭載されるのが

Intel 2012年から（予想）

AMD 2011年から（予想）

Serial ATA 3.0

チップセット内蔵でマザーボードに搭載されるのが

Intel 2011年から（予想）

AMD 2010年から

PCI-Express 3.0

2011年に仕様決定と思われるのでその後
