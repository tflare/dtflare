Operaクッキーの扱い

ブラウザを使っていると使用しないクッキーがたまってくる。

昔は時々削除していたけど

以下の設定により削除する必要がなくなった。

Tools > Preferences > Advanced > Cookies > Delete new cookies when exiting Operaをオンにして

クッキーが必要なサイトで右クリックからEdit Site Preferences

Cookies > Delete new cookies when exiting Operaをオフにする。

すると必要なサイトだけクッキーを受け入れることができる。

そのためクッキーを削除する必要がなくなる。

Ubuntu 9.04アップグレード

Ubuntu 8.10から9.04へのアップグレード時の設定等のメモ

ネットワーク

ネットワークがつながらない。（回線はBフレッツです。）

ppoeconfしてみる→設定が残っている→そのまま進めるとネットワークにつながるようになった。

これは起動時につながらないようになっているだけ？

と思って

sudo gedit /etc/rc.localに以下を記述した所、自動的に接続されるようになった。 

ifconfig eth0 up
pon dsl-provider

opera

プラグインの設定を再設定する必要があった。

numlockの自動有効化を再設定

sudo gedit ‘/etc/gdm/Init/Default’

exit 0の上に以下を追記 

if [ -x /usr/bin/numlockx ]; then
/usr/bin/numlockx on
fi

sources.list

sudo gedit /etc/apt/sources.list

deb http://deb.opera.com/opera stable non-freeのコメントを外す

個人的な改善点

起動時に時々デスクトップのアイコンが表示されないことがあった。

それが改善された気がする。

起動してログアウトして再度ログインすると直っていたので大きな問題ではなかったのだが

改善されたようでうれしい。

冷蔵庫買い換えで節約できるか

三菱電機MR-15Xの年間電気代は14,490円（参考）

これをシャープSJ-17R-Wに買い換えると年間電気代は8,100円（参考）

SJ-17R-Wの最安値は40,500円（参考）

40500÷(14490-8100)≒6.338

6年4ヶ月使うと電気代が冷蔵庫代を上回る

SANYO SR-111Rだと年間電気代は6,210円（参考）

SR-111Rの最安値は27,720円（参考）

27720÷(14490-6210)≒3.3478

3年4ヶ月使うと電気代が冷蔵庫代を上回る

これだと買うのは難しい

SR-111Rでも3年後に新しい冷蔵庫を買った方が節約できるだろうし

後はセール等で冷蔵庫代が安いところを狙えば節約になるかもしれないので

ちょっと注視しよう
