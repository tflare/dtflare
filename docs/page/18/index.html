zsh

chsh

でログインシェルを/bin/zshに変更。

いまさらながらとっても便利

.zshrcを以下に設定

設定内容は以下を参照するとわかりやすい。

http://www.ayu.ics.keio.ac.jp/~mukai/translate/zshoptions.html 

# compinitの初期化
autoload -U compinit
compinit
# 補完候補カーソル選択
zstyle ':completion:*:default' menu select=1
## ヒストリ関連
# 履歴ファイル
HISTFILE="$HOME/.zsh_history"
HISTSIZE=100000
SAVEHIST=100000
# 時刻を記録
setopt extended_history
# 履歴をインクリメンタルに追加
setopt inc_append_history
# 履歴の共有
setopt share_history
# 直前と同じコマンドラインはヒストリに追加しない
setopt hist_ignore_dups
##　プロンプト関連
setopt prompt_subst
local BLUE=$'%{\e[34m%}'
local DEFAULT=$'%{\e[m%}'
PROMPT=$BLUE'${USER}@${HOST}%(!.#.$)'$DEFAULT
RPROMPT='[%~]'

簡易スパム対策

自分が使っている簡易スパム対策

1.知合いやメーリングリスト等の定期的にメールが来るものをメーラの機能で振り分けしておく。

2.spamフォルダを作成する

3.振り分けの設定でToが自分のメールアドレスと一致しない場合にspamフォルダに移動するようにする。

　その際優先度を1で設定したルールよりも低くする。

これだけで8割程度のspamをspamフォルダに自動的に移動できる。

あとは1で振り分けられなかった個人的に来たメールをspamフォルダから移動しspamは削除すればOK
