Bフレッツからauひかりに変更

Bフレッツからauひかりに変更

Ubuntu9.10での設定変更

buntu 8.10から9.04へのアップグレード時の設定を一応消しておく

http://d.tflare.com/2009/04/25/

sudo gedit /etc/network/interfacesで

以下以外がPPPoE関連の設定だったのでコメントアウト

auto lo

iface lo inet loopback

再起動

完了これだけでつながった。
